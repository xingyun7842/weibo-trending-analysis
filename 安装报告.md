## 使用方法

所有插件现已全局安装，无需在每个项目中单独安装。插件会自动在 Claude Code CLI 中可用。

### 查看可用插件
在 Claude Code CLI 中输入：
```bash
/plugin
```

### 查看特定插件的详情
例如查看 Python 开发插件：
```bash
/plugin info python-development
```

### 使用插件中的 Skills
插件中包含的 skills 现在也全局可用。例如：
```bash
/skill python-pro
```

## 注意事项

1. **全局可用**: 所有 70 个插件已安装到全局配置目录，在所有项目中都可以使用
2. **无需重新安装**: 下次启动 Claude Code CLI 时，这些插件会自动加载
3. **更新插件**: 如需更新插件，可以重新运行 `install_all_plugins.py` 脚本
4. **跳过重复**: 脚本会自动跳过已安装的插件，避免重复安装
